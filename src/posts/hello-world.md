## Hello World!

Welcome to my new blog. I've built this using **SvelteKit** and **mdsvex**, which allows me to write posts in Markdown while still being able to use Svelte components inside them.

### Why SvelteKit?

- **Fast**: It's incredibly fast out of the box.
- **Easy**: The developer experience is top-notch.
- **Flexible**: I can mix static and dynamic content easily.

```clojure
(defn greet [name]
  (println (str "Hello, " name "!")))

(greet "Reader")
```

Thanks for reading!
